# قالب إسكان — ملاحظات القالب

## نظرة عامة
- **اسم القالب:** قالب إسكان
- **المسار:** `public/iskan/`
- **مشتق من:** قالب سند (`public/sanad/`) — نسخة مستقلة لا تؤثر على الأصل.
- **القطاع المستهدف:** جمعيات التنمية والإسكان، الإسكان التنموي، ترميم المنازل،
  دعم الأسر الأشد احتياجًا سكنيًا، المبادرات السكنية، والتمكين الأسري المرتبط
  بالاستقرار السكني.

## مبادئ الاشتقاق
- **بنية الملفات:** مطابقة لقالب سند بالكامل (نفس الصفحات، نفس المجلدات،
  نفس ترتيب CSS/JS/Images).
- **أسماء الكلاسات:** محفوظة بدون تغيير. كلاسات مثل `s-hero`, `s-service`,
  `sanad-navy`, `sanad-emerald`, `sanad-gold`… تبقى بنفس الأسماء حفاظًا على
  توافق `JavaScript` و`Skeleton Loading` والمكوّنات.
- **التغييرات المسموحة فقط:** الألوان، النصوص، الأيقونات، الصور، ومحتوى
  بعض الأقسام البصرية (الهيرو على وجه الخصوص).

## الصفحات المتاحة
نفس صفحات قالب سند بدون استثناء، منها:
- الرئيسية، عن الجمعية، الاستراتيجية، التأسيس، الحوكمة، الهيكل التنظيمي،
  مجلس الإدارة، الإدارة التنفيذية، اللجان، الفروع.
- الخدمات وتفاصيلها، المشاريع وتفاصيلها، الأخبار، المركز الإعلامي،
  الفعاليات (بحضوري/أونلاين)، المقالات.
- بوابة المستفيدين بالكامل (دخول/تسجيل/لوحة تحكم/منتجات/طلبات/تتبع/دفع).
- التبرعات وتفاصيلها وسند التبرع، المتجر وسلة الشراء والدفع وفاتورة الطلب،
  العضوية، التطوع والوظائف، اتصل بنا، الشروط والخصوصية.

## محتوى الإسكان
- الهيرو الرئيسي يعكس الاستقرار السكني وتمكين الأسر.
- بطاقات الإحصاء: الأسر المستفيدة، المنازل المرممة، المشاريع المكتملة،
  الشركاء.
- الخدمات والمشاريع وفرص التبرع وبوابة المستفيدين تتمحور حول: ترميم،
  تأثيث، إيجار طارئ، صيانة، استشارة إسكانية، تمكين أسري، وقف إسكاني.

## ضوابط الصور والخصوصية
- استخدم صور منازل، مشاريع عمرانية هادئة، فرق ميدانية، أدوات ترميم،
  مفاتيح، مخططات.
- تجنّب صور الحالات الإنسانية المحرجة أو التي تكشف هوية المستفيدين.
- لا تستخدم صورًا تجارية فاخرة (شركات عقار) ولا صورًا عامة لا علاقة لها
  بالقطاع.

## ملاحظات تقنية
- HTML + Tailwind (CDN) + JavaScript — بدون Bootstrap وبدون أطر React/Vue.
- RTL بالكامل، نصوص الواجهة بالعربية فقط.
- لم تُضَف أي مكتبات أو سكربتات جديدة. `assets/js/main.js`
  و`assets/js/sanad-skeleton.js` تعمل كما في قالب سند.
- Skeleton Loading يحتفظ بكلاساته وبنيته، الألوان فقط تتبع الهوية الجديدة
  عبر متغيرات CSS.

## التفعيل في إدارة القوالب
يُفعّل القالب كخيار مستقل باسم **«قالب إسكان»** ويشير إلى مجلد
`public/iskan/`. الصفحة الافتراضية: `index.html`.
