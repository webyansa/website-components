رجوع

قواعد تشغيلية مهمة

  • الطلب لا يعني اشتراكًا نشطًا — الاشتراك يبدأ فقط بعد التفعيل الرسمي.
  • الدفع المؤكد لا يعني بدء مدة الاشتراك، بل يسمح فقط ببدء التجهيز الفني.
  • لا يمكن تفعيل الاشتراك قبل تأكيد الدفع وقبل اكتمال جاهزية الموقع.
  • تاريخ بداية الاشتراك = تاريخ التفعيل، وتاريخ النهاية = تاريخ التفعيل + مدة الباقة.
  • الفواتير تصدر خارج النظام حاليًا، ويُسجَّل داخل النظام مرجع الدفع فقط.
  • يتم تنفيذ تفاصيل التجهيز الفني في النظام التشغيلي المعتمد (خارج لوحة الاشتراكات).
  • لوحة الاشتراكات تسجل حالة التجهيز فقط ولا تدير المهام الفنية التفصيلية.
