

## خطة التحديثات المطلوبة

### 1. دعم Bootstrap 5 بدون التأثير على التصميم الحالي

- إضافة CDN لـ Bootstrap 5 CSS و JS في `<head>` و قبل `</body>`
- إضافة Bootstrap RTL bundle للدعم العربي
- تغليف الأقسام بكلاسات Bootstrap (`container`, `row`, `col-*`) مع الحفاظ على الكلاسات المخصصة الحالية
- إضافة CSS overrides لمنع Bootstrap من تغيير التصميم الحالي (إعادة تعيين بعض القيم الافتراضية لـ Bootstrap التي تتعارض مع التصميم)

### 2. قسم "انضم معنا" — تعديل الشبكة إلى `col-6 col-6`

- تغيير `.op-joinus-grid` لاستخدام `row` من Bootstrap مع `col-md-6` لكل بطاقة
- إزالة `max-width: 900px` ليأخذ البطاقات عرض الصف كاملاً
- تحديث responsive لـ `col-12` على الموبايل

### 3. شركاء النجاح — سلايدر متحرك

- تحويل `.op-partners-grid` إلى سلايدر أفقي يتحرك تلقائيًا (infinite loop)
- استخدام CSS animation مع `@keyframes` لتحريك الشريط بشكل مستمر
- مضاعفة عناصر الشركاء لتحقيق تأثير التكرار اللانهائي
- إيقاف الحركة عند hover

### 4. فريق العمل — سلايدر متحرك

- تحويل `.team-grid` إلى سلايدر أفقي مع أزرار تنقل (سهم يمين/يسار)
- إضافة نقاط تنقل (dots) أسفل السلايدر
- عرض 4-5 بطاقات في كل مرة مع إمكانية التمرير
- التمرير التلقائي مع إيقاف عند hover
- دعم السحب باللمس على الموبايل

### التفاصيل التقنية

**الملفات المتأثرة:**
- `public/one-page.html` — إضافة Bootstrap CDN، تعديل HTML لأقسام الشركاء والفريق وانضم معنا
- `public/css/one-page.css` — إضافة أنماط السلايدر، overrides لـ Bootstrap، تعديل شبكة انضم معنا

**هيكل سلايدر الشركاء:**
```text
┌──────────────────────────────────────────────────┐
│  ← [شعار1] [شعار2] [شعار3] [شعار4] [شعار5] →   │
│     ══════════════════════════════════            │
│     (تمرير تلقائي مستمر — يتوقف عند hover)        │
└──────────────────────────────────────────────────┘
```

**هيكل سلايدر الفريق:**
```text
┌──────────────────────────────────────────────────┐
│  ◄  [بطاقة1] [بطاقة2] [بطاقة3] [بطاقة4]  ►      │
│                  ● ○ ○                            │
│     (أسهم تنقل + تمرير تلقائي)                    │
└──────────────────────────────────────────────────┘
```

**ملاحظات مهمة:**
- Bootstrap سيُضاف قبل ملف `one-page.css` ليتم تجاوز أي تعارضات
- التصميم الحالي محمي بكلاسات مخصصة ذات أولوية أعلى
- السلايدرات ستُبنى بـ Vanilla JS + CSS بدون مكتبات إضافية للحفاظ على خفة الصفحة
